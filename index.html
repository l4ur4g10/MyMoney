O que é o MiniMyMoney?

É uma versão simplificada do MyMoney, um site/app de gestão financeira aos moldes contábeis que tem e planeja ter integração com diversos bancos. Não tem armazenamento de dados, não tem integração com banco de dados e é completamente manual.

A criação desse site é apenas para fins de aprendizado.
